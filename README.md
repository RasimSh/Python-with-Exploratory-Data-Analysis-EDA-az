# Python ilə EDA Layihəsi

Bu layihə Python vasitəsilə verilənlərin ilkin araşdırılması (Exploratory Data Analysis - EDA) 
üzərində qurulub. Məqsəd verilənlərin strukturunu anlamaq, statistik göstəriciləri çıxarmaq 
və vizual analiz aparmaqdır.


## Məqsəd
* Verilənlərin yüklənməsi və təmizlənməsi
* Statistik göstəricilərin hesablanması
* Vizual analiz (histogram, boxplot, heatmap və s.)
